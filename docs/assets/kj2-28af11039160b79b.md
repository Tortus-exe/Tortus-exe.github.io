# は and が

Last time, we touched upon the details of how <span style="color:#fcd135;">は</span> and <span style="color:red;">が</span> mark two different things. Specifically, <span style="color:#fcd135;">は</span> marks the *focus*, whereas <span style="color:red;">が</span> marks the *doer*.

Since <span style="color:#fcd135">は</span> does not affect the logic of the sentence, we refer to it as a "**non-logical particle**". <span style="color:red;">が</span> marks the doer of the sentence, which if it changes will change what is happening in the sentence.
